# Spring MVC 4 Hello World Example with XML Configuration, Maven and JSP

## Prerequisites
- JDK 1.8 or later

- Maven 3 or later

## Stack
- Spring MVC 4

- JSP View Template

## Maven Run
`mvn jetty:run`

## Docker Run
`docker run -d -p 9999:9999 itnove/simple-maven-spring-boot-example`

## Hello World Web
`http://0.0.0.0:9999/hello`
